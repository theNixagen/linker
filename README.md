# Linker

O Linker é uma API para um app clone do Linktree/Beacons. Inicialmente, eu quis criar esse app para praticar Go e aprender Vue, já que é bem simples desenvolver um MVP desse tipo.

Depois de terminar a API em Go, decidi transformar o projeto em algo maior: criar uma API equivalente em todas as linguagens com as quais eu já havia construído pelo menos uma API.
